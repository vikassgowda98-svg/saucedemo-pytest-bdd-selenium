# SauceDemo Automation | Python + Selenium + Pytest-BDD

This repository contains an end-to-end UI automation framework for the [SauceDemo](https://www.saucedemo.com/) e-commerce application, built using:

- **Python**
- **Selenium WebDriver**
- **Pytest**
- **Pytest-BDD (Gherkin syntax)**
- **Page Object Model (POM)**
- **HTML test reports**
- **GitHub Actions (CI)**

The goal of this project is to demonstrate a real-world automation framework that can be shown in interviews.

---

## Tech Stack

- **Language:** Python 3.x  
- **Test Runner:** Pytest  
- **BDD Framework:** pytest-bdd  
- **UI Automation:** Selenium WebDriver  
- **Design Pattern:** Page Object Model (POM)  
- **Reports:** pytest-html  
- **CI/CD:** GitHub Actions
